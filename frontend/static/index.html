        Estratégia do projeto

        Gestão de Suprimentos — Aplicação Full Stack

        ObjetivoDigitalizar o fluxo de solicitações, aprovações, compras e entregas.
        DesenvolvimentoFastAPI, JavaScript, Tailwind CSS e APIs REST.
        DadosPostgreSQL, SQLAlchemy, migrações e indicadores de custos.
        Data Science/BIDashboards interativos, comparativos e projeções financeiras.
        SegurançaAutenticação, perfis de acesso e validações no backend (OAuth2 com JWT Bearer).
        EngenhariaRastreabilidade de operações, logs e anexos.
        EstratégiaDesenvolvimento incremental orientado aos fluxos reais dos usuários.
        DeployAplicação versionada com Git e publicada no Render.
